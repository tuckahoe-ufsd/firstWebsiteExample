    Welcome to my website. I hope together using this we will gain more understanding about how web design works. HTML is a quite an interesting technology that is considered one of the "core" web technologies. Along with CSS and Javascript you can make very exciting websites.

    Why do we need to use HTML?

    HTML is needed because when we put content on the internet we really arent creating it. HTML is a markup language, when you want to go to a URL your browser requests the website. Then the browser accesses the HTML file and parses or translates the HTML into a DOM (more on that later). After the DOM is constructed the browser renders the HTML and CSS files in order to accurately display the website.

    Is HTML programming

    The shortest answer is no, HTML will prepare you for programming, but its technically not a programming langauge. HTML is missing several features to be considered a coding language, such as logic handling, no ability to store variables, dependent on variables etc. Learn more here.
